Next up, GitHub has announced it will use private code from user repositories to train its AI tools like Copilot, unless users opt out by April 24. This means private code, which was only seen by you or your team, could now help train GitHub’s AI. To stop this, you must go into your GitHub settings and opt out before the deadline. If you do nothing, GitHub will use your code for training. The article explains how to check your settings and why you might want to opt out, such as to protect company secrets or sensitive information. Some companies may not even know about this change if their admins do not check the settings. People in the comments have mixed opinions. Some believe this could break trust with developers and companies, and that GitHub should have asked for permission, not just set it as default. There are also legal worries, since some code has licenses that do not allow this use. A small group thinks it will help AI tools get better and is not a big problem. Some users say companies should leave GitHub if they do not like the change. Others share tips for opting out or checking with their admins. Most agree that GitHub should have explained this more clearly and made opt-in the default, not opt-out.

Petrol stations must make sure buyers follow the rules. The government also wants to make it harder for foreign drivers to buy large amounts. Prices in Austria are much higher than in Slovenia, but Slovenia says their prices will go up soon too. Some lorry drivers in Slovenia are shocked and confused, saying empty petrol stations make it feel like “war.” Slovenians have mixed feelings about the visitors. Some are annoyed by queues and shortages, while others like that “fuel tourists” spend money in shops and restaurants. In the comments, people debate if rationing is a good move or just a short-term fix. Some say it protects local supplies, while others think it will not solve the main problem of high demand and low supply. There are worries that price controls lead to these problems, and some people remember similar fuel limits in their own countries. Some believe governments should let the market decide prices, while others worry that would hurt poor people most. Many think Slovenia’s rules are just a temporary answer, and Europe needs bigger changes to avoid future shortages.

The next article explains the mysterious `.claude` folder used with Claude Code, an AI tool for coding. Many users see this folder in their projects but do not know what the files do. The `.claude` folder is like a control center for Claude. There are two main folders—one for the project and one just for you. The project folder goes into git and sets team-wide rules, while the home folder saves your personal preferences and history. The key file is `CLAUDE.md`, where you set coding rules and style preferences. You can also have personal rules in `CLAUDE.local.md`, which are not shared in git. For big teams, you can split rules into smaller files and make some rules apply only to certain folders. You can add custom slash commands, and even create skills that run automatically. Agents can also be set for focused tasks like code review. There is a `settings.json` file for what Claude can and cannot do, with a personal override file too. The article suggests starting simple and only adding more as your project grows. In the comments, some like the clear structure and control, while others worry about complexity and possible security risks. Some praise path-scoped rules, and others worry the folder could become a mess over time. There are also debates about how much control is good, and if this setup is too much for small projects.

Moving on, Velxio is a new tool that lets you emulate Arduino, ESP32, and Raspberry Pi 3 boards in your web browser. You can write code, add electronic parts, and test hardware projects without buying any devices. Velxio supports 19 different boards and five CPU types, including AVR, ARM, and RISC-V. You can use C++ or Python, compile your code, and see real-time output and errors. There are more than 48 electronic parts you can use, like LEDs, sensors, and displays. You can save and share projects, run Velxio online or on your own computer, and even connect multiple boards together. It uses modern tools like React and FastAPI, and emulation tools like QEMU. Velxio is open-source for personal and educational use, but you need a paid license for business use. On Hacker News, people are impressed by how many boards and features it supports, especially live emulation for Raspberry Pi 3 and real Linux. Some like that it is open-source and self-hostable. There are comparisons to similar tools, and some users wonder about the accuracy of the emulation. Others praise the active community and Discord support, while some report good speed and others have issues on older computers. Most agree Velxio is a big step forward for learning and testing embedded systems.

Now for some important security news. The Telnyx Python SDK package on PyPI was hit by a supply chain attack on March 27, 2026. Two fake versions, 4.87.1 and 4.87.2, with bad code were uploaded and stayed online for about six hours before removal. Only the Python SDK on PyPI was affected. The main Telnyx platform and APIs are safe. If you installed or updated the telnyx package during that time, you may be at risk. You should downgrade to 4.87.0, change all your secrets, and check if your system connected to the attacker’s server. The malicious versions were removed quickly, and Telnyx is investigating how their credentials were stolen. This is part of a bigger wave of attacks, with other Python packages also hit recently. In the comments, people are worried about the growing number of supply chain attacks on package registries. Developers say it’s hard to trust open source packages, and many suggest pinning versions and using tools to check for changes. Some blame weak security around publishing keys. Others say even big companies get hit. There are calls for more checks or two-factor authentication for uploads. Some people consider private package mirrors, but say it’s too much for small projects. Most agree that supply chain risks are here to stay and everyone needs to be careful.

The next article explains how to set up a real HTTPS certificate from Let’s Encrypt on a Brother printer using Certbot and Cloudflare. The writer wanted their printer’s web page to use a trusted, safe connection, like real websites, but without too much manual work. They use Cloudflare for DNS, NextDNS for local names, and Tailscale in their homelab. Brother printers can’t run Tailscale, so they use DNS tricks to reach the printer only from their network. They avoid using a reverse proxy to keep things simple. The setup needs a Brother printer, a computer to run scripts, and DNS records managed by Cloudflare. Certbot is used to get certificates, and a tool called “Brother Cert” uploads the certificate to the printer. The script asks Let’s Encrypt for a certificate, then uploads it to the printer and reboots it. Credentials are stored in separate files for safety. The script can run on a schedule to keep the certificate up to date. In the comments, readers find the solution clever and practical. Some point out that printer security is often ignored, and real certificates are a good idea. Others worry about storing credentials, but the article gives tips for safety. Some ask if this method works for other printers, and some suggest using internal certificate authorities instead. Most agree it’s best to keep printers only available in the local network. Readers like the automation and clear steps, and some share how they might use the script for their own printers.

Our final story is about looking at sand grains under a microscope and discovering how different and beautiful each grain is. Sand is not just tiny rocks—it’s a mix of minerals, shells, and bits of marine life, each with its own story. Grains of sand are between 0.02 and 2 millimeters in size. They come from rocks, shells, and sea life breaking down over time. Each beach has different sand, based on local rocks, volcanoes, and sea animals. For example, sand in Iceland is black, while tropical beaches can be white from coral and shells. There are interactive maps to see sand samples in 3D. The article explains that sand shows the history of its area, shaped by volcanoes, sea, and plate tectonics. In the comments, many people are surprised by how beautiful sand is up close. Some say they never thought sand could be so interesting. Others talk about collecting sand or how sand is made. Some mention that sand is important for making glass and building, so understanding it matters. There are also worries about sand mining and the environment. Some users talk about using microscopes at home to look at everyday things, which makes science more fun. Most readers enjoy seeing how even simple things like sand can be full of hidden detail and wonder.
