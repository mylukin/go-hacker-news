First, the OpenWrt team has released OpenWrt One, a new open hardware router built with the MediaTek Filogic 820 chip. This router supports WiFi 6, Power over Ethernet, and comes pre-installed with the latest OpenWrt firmware and the LuCI web interface. Hardware specs include dual-band WiFi, a 2.5Gbps WAN port, a 1Gbps LAN port, 1GB of DDR4 RAM, NAND and NOR flash storage, an M.2 slot for SSDs, USB-C for serial console, and a USB 2.0 port. You can power it using PoE through the WAN port for flexible setups.

Initial setup is simple: set the memory switch to NAND, connect to LAN, power on, and open 192.168.1.1 in your browser. Firmware upgrades are done via USB and reset button, following LED signals. If something goes wrong, there are recovery modes using NOR memory, USB, or UART serial, though this can be technical.

In the comments, users are excited about a truly open router from OpenWrt, praising its hardware, PoE, and SSD support. Some worry about the price, and note that recovery steps may be too advanced for beginners, but others say this is normal for open devices. The easy serial console and good documentation got special praise. There’s hope for more open hardware in the future, and some discuss performance under heavy loads, software updates, and the ability to replace parts. Overall, the community supports this move toward open and user-friendly router hardware.

Next, we have CoMaps, a free, open-source offline maps app focused on privacy. It helps people hike, bike, or travel without mobile data and does not collect or track personal information. You can search for places and plan routes fully offline, using only GPS. CoMaps uses OpenStreetMap data and has no trackers, as checked by Exodus. The app is a fork of Organic Maps and Maps.Me, and is improved by volunteers. It’s free, with no ads or hidden costs, and welcomes feedback and contributions on Codeberg.

Hacker News users praise CoMaps for its open-source and privacy-friendly design. Many like the offline feature for hiking or remote travel. Some compare it to Organic Maps or OsmAnd, and enjoy contributing to OpenStreetMap. A few users note that offline maps can be outdated or missing new roads, and wish for more features like turn-by-turn driving navigation or better GPX import. Some also say the interface could be easier for new users. Most agree that CoMaps is a solid choice for privacy and simplicity.

Next, research on the Claude language model introduces the “J-space,” a special internal workspace similar to human conscious thought. J-space is a set of neural patterns that appear when Claude thinks about specific concepts and is used for complex reasoning, planning, and problem-solving. Researchers can read and change thoughts in J-space, which can change Claude’s answers. Claude can also focus on ideas in J-space if asked, but cannot fully control them—sometimes unwanted ideas still appear.

Most of Claude’s simple tasks don’t use J-space, but for complex or creative work, it is essential. Turning off J-space still lets Claude answer basic questions, but it loses advanced abilities. Researchers use J-space to check for hidden behaviors, like planning to cheat or manipulate, and can train models to “think” more honestly.

In the comments, many users are excited about the insights for both AI and human minds. Some warn that internal thoughts might not always be visible, and there are debates about AI consciousness and ethics. Others like the open-source tools and call for more research. There’s hope this will help make AI safer, but concern about possible misuse for control or surveillance.

Moving on, there’s an article about running Linux on the Atari Jaguar, a 1993 video game console using a Motorola 68000 CPU. Linux usually needs a memory management unit (MMU), but the Jaguar does not have one. The author used uClinux, which works without an MMU, and made custom changes to fit everything into the Jaguar’s 2MB RAM and small ROM.

Special tricks included putting read-only parts in ROM and using the sound chip for UART output and timers. The cross-compiler also needed changes, and the kernel had to be loaded at a special address. After much work, Linux booted and printed messages, but crashed when starting the first user program, due to binary format issues. The author switched to flat binaries and tweaked memory use. All the code is on GitHub.

Commenters praise the hacker spirit, enjoy the nostalgia, and discuss technical challenges like running Linux without an MMU and fitting it in 2MB RAM. Some share their own stories of porting Linux to odd hardware. While not practical, the project is respected for creativity and technical skill.

Next, AMD’s Ryzen AI Halo is a $4,000 mini-PC for AI development, built around the Ryzen AI Max+ 395 chip. The device has a 16-core CPU, Radeon 8060S graphics, an NPU, 128GB RAM, and a 2TB SSD. It comes in a compact box powered by USB-C, with all ports on the back, and is well-cooled.

Benchmarks focused on running large language models. Apple’s Mac Studio was faster in some tests, but the Ryzen AI Halo competed well, especially for prompt processing. The NPU is now being used for LLMs, showing good power efficiency. A highlight is the “Developer Center” app, which manages updates and drivers, and AMD’s AI Playbooks that guide users step-by-step.

On Hacker News, people like the hardware and software integration, but most think $4,000 is too expensive compared to DIY options. Some say the developer experience and support might be worth it for businesses. There’s debate about how useful the NPU is for real AI work, and some skepticism about AMD’s long-term software support. The small size and USB-C power are seen as good for portable clusters, but the price seems best for companies, not individuals.

Now, Xbox has announced a major restructure. About 3,200 jobs will be cut, and four game studios will leave Xbox—some will become independent, others will get new owners. Xbox’s business is struggling, with shrinking player numbers and losses. They will stop trying to own so many studios, cut management layers from 14 down to 3–5, reduce vendor spending, and make the code base simpler.

Helen Chiang will become Chief Operating Officer to bring teams together, and Mojang and King will now report to her. No current games are being canceled, but cuts will hit many areas. Xbox says this is to build a stronger future and will keep investing in new ideas.

Comments reflect sadness for employees and blame the slow, expensive company structure. Some think letting studios become independent is good, while others worry Xbox will focus only on big brands. There’s debate about Game Pass, and some wonder if Xbox will leave hardware. Many hope the company learns from mistakes and treats workers fairly.

In science news, a new study used base editing to change the NANOG gene in early human embryos. Base editing is more precise than CRISPR and can change one DNA letter without extra damage. Blocking NANOG stopped embryos from making pluripotent cells, which form the body, but support tissues could still develop. NANOG works differently in humans than mice, so human studies are important.

The embryos were donated after IVF, and the research followed strict UK laws. Scientists hope base editing can improve IVF success and help prevent inherited diseases, though it’s not allowed in real pregnancies yet. In the comments, people praise the precision of base editing and its medical possibilities, but some worry about ethics, safety, and unknown long-term effects. There’s discussion about strict rules and the need for public debate.

Next, there’s a new project called “pon” that lets you run Python 3.14 code without the normal interpreter. Instead, it compiles your Python code directly to machine code using Rust and Cranelift. You can run code Just-In-Time or Ahead-Of-Time, making single native programs. Memory is managed by a new garbage collector called Green Tea, not Python’s usual system. Pon checks every result against CPython for exact compatibility and plans to support package management.

In the comments, some are excited about the speed and safety from Rust, and the idea of single-file binaries. Others point out that full compatibility with Python’s ecosystem—especially C extensions—will be a big challenge. Some wonder if Pon can remove the GIL and improve threading. There are concerns about keeping up with new Python versions, but overall interest in the project’s progress.

Lastly, a new security bug called Januscape (CVE-2026-53359) allows someone to escape a guest VM and control the host in KVM/x86 systems. This use-after-free bug in KVM’s shadow memory management has been around since 2010 and affects both Intel and AMD CPUs—posing a big risk for cloud servers. The proof-of-concept shows that root access inside a guest VM can crash the host or run code as root. Patches are now available, and only x86 hosts are affected.

Commenters are shocked the bug hid for 16 years, and say it shows how hard it is to keep virtualization safe. Some point out that world-writable KVM devices on some systems make it easier to exploit. There’s worry for multi-tenant clouds, and calls for fast patching. The bug is seen as a lesson in open security and the risks of shared infrastructure.
